Below picture represents layout

Red arrow shows Menubar, opening a photo,editing,croping,photo effect,fonts and more...

Yellow arrow shows "Toolbar,that contains photo processing functionalities like crop,selection,shapes,colors...

White arrow shows Toolbar Options, when you select an option from Toolbar,if selected function has some properties, you can change from this panel...

Blue arrows shows Workspaces, you can create workspaces as many as you want, work separately with them...

Purple arrow shows Layers, you can add lots of layers,reorder ,rename and make them selected...

Black arrow shows Layer Properties, when you select a layer, this panel shows properties of layer like width,height,blending,rotate angle options...
