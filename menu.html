Menu Bar

Lets look at menu bar details

  • With File menu, you can load a photo,image or svg from local computer, from sample images, create a new empty photo, exporting photos as jpg, png or ico format, or preview current work

  • With Edit menu,you can cut,copy,paste or delete photo selected regions, also support undo, redo operations

  • With Workspace menu,you will be able to resize workspace, rotate 90 degrees, flip vertical or horizontal and also change background color

  • With Layer menu, you will be able to create a new layer, copy from selected region of layer, clone selected layer, open a photo layer from a local file, new layer from samples photos, flip photo on selected layer

  • With Filters menu, you will be able to apply photo effects, color adjustments filters like (brightness,contrast,saturation, light), grayscale filters, image sepia filters, image blur filter, image soften filter, image sharpen filter, image edge detection filter, image edge gradient filter, image emboss filter, high pass filter, low pass filter, erode filter, media filter, photo oil painting filter, photo cartoon filter

    First select a layer, then select a filter category from submenus

  • With Font menu, you will be able to add Google fonts

  • With Help menu, you will be able to learn keyboard shortcuts, contact, bug report
