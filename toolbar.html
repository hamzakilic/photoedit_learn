Toolbar

First select a layer from Layers

Lets look at toolbar details

  • after enabling this tool, you will be able to move a selected layer, or resize a selected layer

    for moving a selected layer,click mouse left button then move mouse

    for resizing a selected layer, select an edge and then click mouse left button then move mouse

  • for croping a photo use this tool

    for selecting crop region, first click mouse left button then move mouse then release mouse button

  • for selecting a region of photo, there are 4 selection tools, rectangle selection, ellipse selection,lasso selection, polygonal selection

    for intersection of regions, there a 4 options as intersection,union,self or xor Toolbar Options

  • magicwand selection is based on similar color, just click mouse left button and tool will try to select a region

  • for selecting a color from photo as workspace foreground color, you can use this tool

    after enabling this tool, start with clicking mouse left button, don't release mouse button until finish

  • for brushing a portion of photo, use this tool

    click mouse left button then move mouse over selected layer or selection region

    from toolbar options, you can set size of brush and more ...

  • for erasing a portion of photo, use this tool

    click mouse left button then move mouse over the selected layer or selection region

    from toolbar options, you can set size of erase and more ...

  • for gradient filling a selected layer or selection of a layer

    click mouse left button then move mouse then release mouse button
